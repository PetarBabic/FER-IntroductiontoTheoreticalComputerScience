# FER - Introduction to Theoretical Computer Science

Laboratory solutions for Introduction to Theoretical Computer Science labs during the 23/24 academic year.

Lab1 (Java) - Nondeterministic finite automaton with ε-transitions simulator
Lab2 (Java) - Implementation of deterministic finite automaton (DFA) minimization
Lab3 (Java) - Software implementation of a deterministic pushdown automaton (DPDA)
Lab4 (Java) - Software implementation of a recursive descent parser
